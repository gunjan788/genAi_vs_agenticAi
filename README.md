# 🤖 Generative AI vs Agentic AI

## 📌 Overview

This project demonstrates the key differences between **Generative AI** and **Agentic AI** using real-world task scenarios.

While Generative AI focuses on producing content based on prompts, Agentic AI goes a step further by **planning, reasoning, and executing tasks step-by-step**.

---

## 🎯 Objective

To provide a clear, practical understanding of how AI systems are evolving:

* From simple text generation
* To intelligent systems capable of decision-making and task execution

---

## ⚡ Features

* 🔹 Compare outputs of Generative AI and Agentic AI
* 🔹 Real-world task example (e.g., trip planning)
* 🔹 Step-by-step execution by Agentic AI
* 🔹 Streamlit-based interactive interface
* 🔹 Easy-to-understand comparison

---

## 🧠 Key Difference

| Generative AI        | Agentic AI                  |
| -------------------- | --------------------------- |
| Generates responses  | Plans and executes tasks    |
| Output-focused       | Process + execution focused |
| Single-step response | Multi-step reasoning        |
| No autonomy          | Autonomous decision-making  |

---

## 💻 Tech Stack

* Python
* Streamlit
* OpenAI API
* LangChain

---

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/gunjan788/genAi_vs_agenticAi.git
```

2. Navigate to project folder:

```bash
cd genAi_vs_agenticAi
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
